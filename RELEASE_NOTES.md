## Release v1.0-ready

This release includes the ELCHAPO.dylib file, designed for arm64 architecture on iOS 12.0 and above. Built from the source files contained within EL_CHAPO_build.zip, this dylib incorporates crash prevention features and implements safe hooks to ensure stability and security during performance.

### Additional Notes
- **Architecture**: arm64
- **Minimum iOS Version**: 12.0+
- **Features**:
  - Crash prevention mechanisms
  - Safe hooks for enhanced security
